# Diwali_Sales_Analysis
Analyze customer purchasing behavior during the Diwali festival season in India using sales data to uncover insights about demographics, product preferences, and spending patterns.

#Objectives:
  Explore sales patterns by gender, age group, occupation, and state.
  Identify top-performing product categories.
  Provide actionable insights for business decisions.

#Dataset:
  File: Diwali Sales Data (CSV)
  Columns: User_ID, Gender, Age Group, State, Marital_Status, Occupation, Product_Category, Amount.

#Tools & Tech:
  Language: Python
  Libraries: Pandas, NumPy, Matplotlib, Seaborn
  Environment: Jupyter Notebook.

#Methodology:
  Data import and cleaning.
  Exploratory Data Analysis (EDA) using statistics and visualizations.
  Analysis by demographics, occupation, state, and product category.
  Insights and recommendations.
